# Dev Log 01: Scope

## Objective

将招聘岗位里的 “AI-Native 工作流” 和 “Multi-Agent 协作开发复杂需求” 转化成一个可投递的游戏研发样品。

## Decision

选择 2D 实时俯视角轻量肉鸽，而不是大型关卡、3D 项目或真实在线 AI NPC。

## Reasoning

- 轻量肉鸽有清晰的玩法闭环：移动、战斗、升级、胜负。
- Phaser + React 可以快速交付网页 demo，面试官打开成本低。
- 工作台、AGENTS 协议和 dev-log 能直接展示多 Agent 协作过程。
- 不接真实模型 API，避免现场演示受网络、Key 和费用影响。

## Result

第一版范围固定为：一个实验室主题可玩 demo、一个 AI-Native 研发工作台、核心规则测试和过程文档。
